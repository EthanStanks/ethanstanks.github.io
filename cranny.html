About

In this two-month project, I played a pivotal role in developing Cranny, an Android Studio Kotlin app. In the first month, I focused on creating the social page, including a feed to showcase friends' recently read books and profile information. Additionally, I implemented Firebase's Google authentication for the login page and created classes for seamless CRUD operations on Firebase.

  • Developed the social page with a feed displaying friends' recently read books and profile information
  • Implemented Firebase's Google authentication for the login page
  • Created classes to enable CRUD operations on Firebase, ensuring seamless data management
  • Reworked the layout and features of the social page in the second month
  • Implemented functionality to view friend's reading progress and discover similar books
  • Enhanced the CRUD system for efficiently pulling friends' recently read books
  • Provided extensive debug support to identify and resolve any issues encountered during development
