About

My Vulkan renderer class project is a graphics renderer created in Vulkan using an Object Oriented approach to Data Driven Rendering. The goal is to draw the visual aspects of a game level using the Vulkan API. The project features intuitive controls and a range of impressive capabilities, including support for exporting OBJ names and locations from Blender Game Levels, run-time file I/O parsing of exported level information, and the ability to draw all models successfully at the proper location, orientation, and scale. Additional features include directional light with ambient term and specular highlights, support for hot-swapping multiple levels without a restart, and credits to low-poly models from Quaternius. Overall, my Vulkan renderer class project is a testament to my skills in computer graphics and game development.

  • The project is a graphics renderer designed to draw the visual aspects of a game level using the Vulkan API.
  • It employs an Object Oriented approach to Data Driven Rendering.
  • The project supports exporting one or more Blender Game Levels to a file (.blend) which is parsed at run-time.
  • Exported OBJ names and locations from the levels are read using binary model data and transferred to the GPU for rendering.
  • The project features a working 3D fly-through camera with controls to move the camera in various directions and tilts using the mouse.
  • The project supports loading saved levels using the F1 key, which allows hot-swapping of multiple levels without restarting the application.
  • The project features directional light with ambient term and specular highlights.
  • All models are drawn at the correct location, orientation, and scale.
  • All models have correct OBJ material colors showing.
  • The Dev4 Vulkan Renderer Project is a powerful tool for game developers looking to create high-quality graphics for their games using Vulkan.
