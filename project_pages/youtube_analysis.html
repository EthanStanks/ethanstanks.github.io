YouTube Channel Content Analysis

Description

This Python project is designed to analyze content from multiple YouTube channels. It provides insights into video details, engagement patterns, temporal behaviors, and more. The project consists of various modules to scrape YouTube data, perform data analysis, and print the results.

Details

  • Selenium Web Scrape
  • BeautifulSoup Web Scrape
  • YouTube Data API
  • Pandas
  • Matplotlib Visualizations
  • Research Paper
  • Individual Project
  • One Month Scope

Objectives

  • Scrape data or use web API
  • Store data to a well formatted dataframe, save to file
  • Create at least two (very different) visualizations
  • Aesthetics
  • Analysis
