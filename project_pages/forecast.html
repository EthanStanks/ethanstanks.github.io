Description

As part of a team, I developed a 3D tower defense game using the Unity engine. The game utilizes a unique card conveyor belt and weather system to defend against enemy waves across three distinct maps. My main responsibilities included creating all UI systems and elements, as well as designing and implementing the card system. This card system features a variety of cards, each with different towers, weather effects, buffs, and traps. By incorporating cards, the game provides a range of outcomes for each playthrough and increases its overall replayability.

Details

  • 3D tower defense game using Unity Engine 3D
  • Implemented a card-based gameplay system that features various towers, weather effects, buffs, and traps (C# Script)
  • Created a unique card conveyor belt system that delivers cards to the player for placement on the game board (Unity Animation)
  • Developed a weather system that affects gameplay by altering the game environment and tower effectiveness (C# Script)
  • Programmed game logic and mechanics for player actions, such as tower placement, card selection, and tower upgrades (C# Script)
  • Created all UI systems and elements, including menus, buttons, and on-screen displays (Unity UI)
  • Optimized game performance and memory usage through efficient coding practices and asset management (C# Script)
  • Group Project
  • Three Month Scope (One month planning | Two month development)
