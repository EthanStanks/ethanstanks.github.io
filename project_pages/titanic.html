Titanic Passenger Survival Prediction

Description

This research project is for analyzing survivors of the titanic and creating a basic SKLearn model for making predictions about the survivors. The dataset was provided to the class as this project is meant to be an interesting real world application in my education in data science. My model predicts if a passenger will survive based on the passenger's sex as well as their ticket class. Based on these two inputs a solid prediction can be made about a passenger's survival rate.
