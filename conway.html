Conway's Game of Life



GitHub GitHub Repository


About

This is a Windows C# application that renders Conway's Game of Life, a classic cellular automaton that simulates the life and death of cells on a 2D grid, with additional features such as the ability to turn cells on and off by clicking on them, randomize or empty the universe, save and open previously saved universes, display the current generation and number of living cells, control the speed and size of the universe, and show the neighbor count for each cell, as well as advanced features such as game colors, boundary behavior, context-sensitive menu, a heads-up display, and settings to customize the behavior of the application.

  • The application lets you simulate a popular game called Conway's Game of Life.
  • You can turn cells on or off by clicking on them with the mouse.
  • The game follows certain rules to decide which cells live or die.
  • There are menu items and tool strip buttons for starting, pausing, and advancing the game by one generation.
  • You can randomize or clear the universe, and save or load universes to and from text files.
  • The game displays information about the current generation, living cells, and neighbor count for each cell.
  • You can customize the game's appearance and boundary behavior options.
  • There is a context-sensitive menu for additional options based on your actions.
  • The game shows a heads-up display for current game state information.
  • You can adjust the speed of the game and the size of the universe.
  • The application has settings that you can customize to change the behavior of the game.
